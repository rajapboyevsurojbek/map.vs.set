Map va Set

Map

Map - bu JavaScript’dagi kalit-qiymat juftlari bo‘yicha ma’lumotlarni saqlash imkonini beruvchi kolleksiya.

Map misol:

  • Kalit: "name", Qiymat: "Sirojbek"
  • Kalit: "age", Qiymat: 25

Set

Set - bu faqat noyob qiymatlarni saqlaydigan kolleksiya. Takroriy qiymatlar saqlanmaydi.

Set misol:

  1. Apple
  2. Banana
  3. Apple (takrorlanmaydi)

Ma'lumot: Map va Set turli xil kolleksiya turlarini yaratish va boshqarish uchun ishlatiladi, har biri o‘ziga xos xususiyatlarga ega.
